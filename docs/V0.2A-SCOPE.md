# v0.2A 范围定义

## 已实现

1. 创建口述史项目（项目名称、说明）
2. 创建一位主要讲述者（姓名、关系、出生年份、备注）
3. 记录细粒度同意状态（5 项布尔值 + 确认时间 + 确认方式 + 备注）
4. IndexedDB 本地持久化（Dexie.js）
5. 页面刷新后恢复数据
6. JSON 导出（信封结构）
7. JSON 导入为新项目（不覆盖已有项目）
8. 删除本地项目（transaction 原子删除）

## 边界

- `theme` 字段保留在数据模型定义中但本阶段不使用
- 不修改或迁移比赛稳定版
- 应用运行时不请求任何外部网络资源
