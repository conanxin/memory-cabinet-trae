# V0.2C1 范围说明

## 版本
0.2.0-alpha.3

## 本轮实现范围

### R3C - MemoryItem 视图层

1. **MemoryItem 列表视图** (MemoryItemListView.vue)
   - 显示全部记忆卡片
   - 按状态分组统计（草稿/已确认/已排除）
   - 空状态提示

2. **MemoryItem 创建视图** (MemoryItemCreateView.vue)
   - 从访谈原文选区自动截取证据（sourceStart/sourceEnd）
   - 支持六种卡片类型选择
   - 默认值：sourceType=first_hand, certainty=approximate, visibility=private, reviewStatus=draft
   - 显示项目和访谈上下文信息

3. **MemoryItem 详情视图** (MemoryItemDetailView.vue)
   - 显示完整元数据
   - 原文范围显示（sourceStart - sourceEnd）
   - 来源访谈链接
   - 编辑功能（title、editedText）
   - 状态流转（draft→confirmed→excluded）
   - 删除功能

4. **项目详情视图** 集成
   - 记忆卡片数量统计
   - 记忆卡片列表展示

5. **访谈详情视图** 集成
   - 关联记忆卡片数量统计
   - 关联记忆卡片列表

### 导入/导出
- schemaVersion 3 JSON 导出（含 memoryItems）
- schemaVersion 3 JSON 导入（ID 重新生成，引用重写）
- schemaVersion 1/2 兼容导入

## 明确不包含

- Exhibition 展柜展示功能
- AI 辅助能力
- 图片/音频上传
- 云端同步
- schemaVersion 4
