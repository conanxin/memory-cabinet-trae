# 时光展柜

> 把一段口述变成一座微型人生展览

## 项目简介

**时光展柜**是一款面向家庭记忆保存与代际传承的轻量策展工具。它的核心定位是：把长辈的一段口述记录，转化为一座结构清晰、可浏览、可分享的微型人生展览——包含时间线、地点、人物、旧物、原话摘录和后续访谈追问。

当前为 **TRAE AI 创造力大赛初赛交互原型**。

## 体验方式

打开 `index.html` 即可在浏览器中直接运行，无需服务器或构建步骤。

建议体验路径：
1. 首页浏览预置案例预览
2. 点击「先看看《外婆的缝纫机》」
3. 浏览完整的微型人生展览（时间线、地点、人物、旧物、原话、追问）

新增口述指南：帮助用户从人物、地点、旧物或具体时刻开始讲述。打开 guide.html 即可查看。
